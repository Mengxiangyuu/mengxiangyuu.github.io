Frequency analysis

why？

We can find approximate distribution of data
eg: find the characteristics of data(age sex…)

find the opinion of issue(Does student like this course? A. very like B. like c. don’t like)

How?

分析 → 描述统计 → 频率

words

  1. Inter quartile range : 0% 25% 75% 100%

  2. statistical dispersion: stretched or squeezed

  3. Standard deviation: measure dispersion

  4. skewness: the data is shifted to left or right.

    if skewness > 0;

    shifted to the right;

    if skewness < 0;

    shifted to the left;

  5. kurtosis: Data is flat or steep (>0,steep)
